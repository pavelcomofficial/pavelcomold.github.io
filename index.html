NEW! Мобильное приложение

Зачем бизнесу использовать progressive web apps?

  • Устанавливается на устройство в один клик без магазинов приложений.
  • Быстро загружается благодаря заранее созданному кэшу страниц.
  • Работает офлайн благодаря тому же кэшу.
  • Рассылает пользователям push-уведомления.

PWA или прогрессивное веб-приложение (англ. progressive web app) – гибрид сайта и приложения для мобильных устройств.

В десктопном браузере progressive web app остается обычным сайтом. А когда посетитель открывает его в мобильном браузере, PWA превращается в гибрид сайта и приложения.

Progressive web app используют такие гиганты как, OLX, Uber, AliExpress, Starbaucks, Tinder.
